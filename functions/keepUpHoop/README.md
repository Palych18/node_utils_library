# keepUpHoop

Необходимо реализовать функцию __`keepUpHoop`__.  
Пользователь вводит __(n)__ количество оборотов обруча.  
Функция возвращает сообщение пользователю.  
Если пользователь прокрутил менее 10 раз, то вернется строка:  
__"Keep going until you get it right"__.  
Если пользователь прокрутит 10 или более раз обруч, то вернется строка:  
__"Great, now on to tricks"__.  
Если пользователь вводит не число, то вернется строка:  
__"Enter a number"__.  

```
keepUpHoop(4) // 'Keep going until you get it right';
keepUpHoop(17) // 'Great, now on to tricks';
keepUpHoop('one') // 'Enter a number';
```
